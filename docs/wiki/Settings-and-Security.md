# Настройки и безопасность

## Профиль

В настройках можно:

- загрузить аватар;
- поменять отображаемое имя;
- выйти из аккаунта;
- удалить аккаунт навсегда.

Ограничения для аватара:

- поддерживаются изображения PNG, JPEG, WEBP и GIF;
- максимальный размер примерно 1.5 MB.

## Внешний вид

На мобильных клиентах доступны:

- светлая тема;
- темная тема;
- режим fullscreen.

В браузере и Windows часть визуальных настроек совпадает, но набор элементов оформлен по веб-версии.

## Шифрование сообщений

WAVE поддерживает локальное шифрование текста по ключу Виженера.

Как это работает:

1. Вы включаете `Encrypt send`.
2. Указываете свой ключ.
3. Сообщение шифруется на устройстве перед отправкой.
4. Сервер хранит зашифрованный текст и только помечает тип шифрования.
5. Расшифровка происходит на клиенте.

Что важно:

- это не сквозное шифрование в классическом смысле;
- если у собеседника другой ключ, текст будет выглядеть как набор непонятных символов;
- ключ хранится локально на вашем устройстве.

## 2FA

WAVE умеет работать с Google Authenticator.

### Как включить

1. Откройте `Настройки` -> `Безопасность`.
2. Нажмите запуск настройки 2FA.
3. Отсканируйте QR-код в Google Authenticator.
4. Введите текущий 6-значный код.
5. Подтвердите включение.

### Как входить после включения

1. Введите логин/email и пароль.
2. После этого введите 6-значный код из приложения-аутентификатора.

### Как отключить

- В настройках безопасности введите текущий код 2FA и подтвердите отключение.

## Защита чата

В браузере и Windows есть дополнительная защита чата через 2FA:

- чат можно заблокировать;
- для разблокировки нужен код из аутентификатора.

Это полезно, если вы оставляете открытый аккаунт на общем компьютере.

На мобильных клиентах отдельного интерфейса защиты чата в коде сейчас не видно.

## Звук и устройства

### Браузер и Windows

Можно настроить:

- микрофон;
- динамик;
- громкость;
- тест микрофона;
- звук звонка;
- push-уведомления.

### Android / iOS

Можно настроить:

- громкость микрофона;
- громкость динамика;
- звуки звонков;
- локальное поведение уведомлений.

## Удаление аккаунта

Удаление аккаунта необратимо. Вместе с аккаунтом удаляются:

- профиль;
- связанные переписки;
- серверные данные аккаунта.

Перед удалением убедитесь, что вам действительно это нужно.

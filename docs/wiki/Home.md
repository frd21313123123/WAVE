# WAVE: как пользоваться мессенджером

WAVE это личные и групповые чаты, звонки и базовые инструменты безопасности в одном приложении. Ниже короткая карта, чтобы новый пользователь быстро понял, что где находится.

## Где работает WAVE

| Платформа | Что это значит для пользователя |
| --- | --- |
| Браузер | Полная веб-версия: личные и групповые чаты, звонки, push, PWA, голосовые сообщения, ответы, пересылка, реакции |
| Windows | Это оболочка над веб-версией, поэтому функции почти такие же, как в браузере |
| Android / iOS | Нативный клиент: чаты, группы, профиль, 2FA, шифрование, звонки 1:1; часть веб-функций пока упрощена |

## Если нужно начать за 2 минуты

1. Откройте WAVE и создайте аккаунт или войдите по логину/email и паролю.
2. Нажмите `Новый чат` и найдите пользователя по логину, имени или email.
3. Откройте диалог и отправьте сообщение.
4. Для звонка откройте личный чат и нажмите кнопку аудио- или видеозвонка.
5. Для аватара, 2FA, шифрования и звука откройте `Настройки`.

## Что умеет WAVE

- Личные диалоги и групповые чаты.
- Сообщения в реальном времени.
- Индикатор `печатает...` и отметки прочтения.
- Редактирование своих текстовых сообщений.
- Аудио- и видеозвонки в личных чатах.
- Аватар, отображаемое имя, 2FA и удаление аккаунта.
- Локальное шифрование текста по ключу Виженера.

## Важные отличия между платформами

| Возможность | Браузер / Windows | Android / iOS |
| --- | --- | --- |
| Личные и групповые чаты | Да | Да |
| Аудио- и видеозвонки 1:1 | Да | Да |
| Push-уведомления | Да | Частично: в коде есть локальные настройки и звуки, но не полноценный background push |
| Голосовые сообщения | Да | Можно воспроизводить, но не видно отдельного интерфейса отправки |
| Ответ, пересылка, реакции | Да | Не найден отдельный UI |
| Массовое удаление сообщений / удаление чата | Да | Не найден отдельный UI |
| Защита чата кодом 2FA | Да | Не найден отдельный UI |

## Что важно помнить

- Звонки работают только в личных чатах, не в группах.
- На мобильных клиентах набор функций сейчас уже, чем в веб-версии.
- Если включено шифрование Виженера, собеседник должен использовать тот же ключ, иначе текст будет нечитаемым.
